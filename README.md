```elixir

  [
    {:ocpp_model, "~> 0.1.0"}
  ]
end
```

Have a look at 
```
lib/test/ocpp_charger_test.ex
lib/test/ocpp_chargesystem_test.ex
```
for an example on how to implement a Charger or ChargeSystem with this library
